<h1 style="align: center;">B_MAP_web</h1>
A web app to manage the financial records.

## Requirments
- Ruby version: 2.7.0
- Rails version: 6.0.2
- database: postgresql
-  Minimagick
```
sudo apt-get install -y ruby-mini-magick
```
- GraphicsMagick
```
sudo apt-get install -y graphicsmagick
```
